AvaSpec.lvlib:AvaSpec Main.vit

This VI provides control and an interface for Avantes Spectrometers. It is built based on the queued message handler template and low level function VI's from Avantes. It is made to function as a stand-alone piece of sofware, or to be controlled by the Measurement Control package. In addition to control and logging, it performs standard color metric calclations based on the spectrum it collects.
